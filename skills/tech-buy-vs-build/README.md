# Buy vs. Build

A four-role AI sourcing decision framework. Applies NANDA's 2:1 buy evidence and a vendor-archetype taxonomy to produce a BUILD / BUY / PARTNER recommendation.

## What it does

Maps each AI component being decided, applies the 2:1 buy evidence test, selects the right vendor archetype for BUY decisions, and produces a sourcing strategy that identifies where to build the moat vs. where to buy commodity.

## When to use it

Before any significant AI procurement decision. When debating build vs. buy for AI infrastructure. Before issuing an AI RFP. As part of the pilot-to-production transition when sourcing decisions become binding.

## What it outputs

- Component-by-component BUY / BUILD / PARTNER verdict
- Vendor archetype and candidate vendors for each BUY decision
- Moat assessment (which investment builds defensible advantage)
- Risk flags (misallocated build investments, strategic lock-in risks)
