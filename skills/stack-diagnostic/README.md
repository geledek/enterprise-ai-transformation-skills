# Stack Diagnostic

A six-layer AI technology stack assessment: data, model, orchestration, tools, oversight, operations. Surfaces the weakest link and outputs a prioritized remediation plan.

## What it does

Walks each layer of the enterprise AI stack systematically, scores it (STRONG / ADEQUATE / GAP / BLOCKING), and produces a remediation priority list. The weakest link determines where to invest — usually data or orchestration, not the model.

## When to use it

Before any major AI infrastructure investment. When pilots aren't scaling. When you need to explain why "we have AI but it's not creating value." As an annual stack health check.

## What it outputs

- Six-layer stack scorecard
- Weakest-link identification with explanation
- MIT CISR maturity stage implication
- Remediation priorities ordered by dependency
